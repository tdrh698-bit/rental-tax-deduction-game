# 住房租金扣除游戏 - 完整包

## 游戏简介
这是一个教育性的住房租金扣除游戏，包含4个任务关卡，帮助用户学习中国个人所得税住房租金扣除政策。

## 游戏任务

### 任务一：城市等级判断
- **目标**：将8个城市拖拽到正确的扣除标准区域
- **城市**：北京市、厦门市、杭州市、苏州市、温州市、佛山市、伊春市、七台河市
- **扣除标准**：1500元/月、1100元/月、800元/月

### 任务二：租赁合同审查
- **目标**：找出租赁合同中的5个问题点并修复
- **技能**：合同审查、问题识别

### 任务三：扣除计算实战
- **目标**：根据城市和实际租金计算可扣除金额
- **功能**：城市选择、租金滑块、实时计算

### 任务四：申报材料准备
- **目标**：在60秒内收集所有必需材料
- **材料**：8种材料卡片，区分必需和可选

## 快速启动

### 方法一：使用Python HTTP服务器
```bash

# 启动HTTP服务器（端口3000）
```

## 城市数据

游戏使用8个指定城市：
- **1500元/月**：北京市（直辖市）、厦门市（计划单列市）、杭州市（省会城市）
- **1100元/月**：苏州市、温州市、佛山市（人口>100万）
- **800元/月**：伊春市、七台河市（人口≤100万）
